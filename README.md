# Number Guessing Game

## Project Description

This is a simple Python-based Number Guessing Game.

The computer randomly generates a number between 1 and 100. The player has to guess the correct number. After each guess, the program provides hints such as "Too High" or "Too Low" until the correct number is guessed.

## Features

* Random number generation
* User input handling
* Hint system (Too High / Too Low)
* Attempt counter
* Beginner-friendly Python project

## Technologies Used

* Python 3

## Concepts Learned

* Variables
* Loops
* Conditional Statements
* User Input
* Random Module
* Problem Solving

## How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the following command:

```bash
py guessgame.py
```
