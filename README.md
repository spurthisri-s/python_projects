# Python Projects

This repository contains Python mini-projects developed while learning Python programming, problem-solving, and software development fundamentals.

## Projects

### 1. Number Guessing Game

A game where the computer generates a random number and the user tries to guess it.

**Concepts Used**

* Variables
* Loops
* Conditions
* Random Module
* User Input

---

### 2. Student Marks Report Card

A program that accepts student details and marks, then calculates the grade.

**Concepts Used**

* Variables
* Input and Output
* If-Elif-Else
* Conditional Logic

---

### 3. Multiplication Table Generator

Generates the multiplication table of a given number.

**Concepts Used**

* For Loops
* User Input
* Mathematical Operations

---

### 4. Calculator Using Functions

A calculator that performs arithmetic operations using Python functions.

**Concepts Used**

* Functions
* Parameters
* Return Values
* Arithmetic Operations
